# Spacex Launch Manifest

A new Flutter project.

## About

This project shows list of Rockets designed, developed and manufactured by SpaceX and its manifests.

To run this project

- `install latest flutter SDK`
- Clone this repo
- `flutter pub get`
- `flutter run`
